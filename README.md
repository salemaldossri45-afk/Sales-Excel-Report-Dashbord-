Excel Sales Analytics Dashboard

Project Overview

This project was developed using Microsoft Excel to analyze sales performance, profitability, products, cities, and sales channels. The dashboard provides a clear overview of business performance through interactive reports and visualizations.

Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design
* Data Analysis

Key Metrics

* Total Sales
* Total Profit
* Profit Margin
* Quantity Sold
* Product Performance
* City Performance

Dashboard Features

* Sales Analysis by Product
* Monthly Sales Trends
* Profitability Analysis
* City-Level Sales Analysis
* Sales Channel Analysis
* Interactive Dashboard Reporting
